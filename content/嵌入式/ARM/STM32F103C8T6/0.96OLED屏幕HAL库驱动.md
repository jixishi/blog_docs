## 起因

HAL库的资料相对于标准库来说还是较少，而且绝大部分教程还在csdn这个平台上，资源质量不稳定，难以快速使用，所以便有了此多兼容驱动库

### 库实现

|      函数名      |      功能      |
| :--------------: | :------------: |
|     OLED_CLS     |      清屏      |
|    OLED_Fill     |    全屏填充    |
|  OLED_ColorTurn  |    反色显示    |
| OLED_DisplayTurn |    反转屏幕    |
|   OLED_Refresh   |    刷新屏幕    |
|  OLED_DrawLine   |      画线      |
| OLED_DrawCircle  |      画圆      |
| OLED_ShowString  |   显示字符串   |
|  OLED_ShowFloat  | 格式化浮点显示 |
| OLED_ShowPicture |  显示BMP图片   |

    }

void OLED_DrawCircle(uint8_t x,uint8_t y,uint8_t r)
{
    int a, b,num;
    a = 0;
    b = r;
    while(2 * b * b >= r * r)
    {
        OLED_DrawPoint(x + a, y - b);
        OLED_DrawPoint(x - a, y - b);
        OLED_DrawPoint(x - a, y + b);
        OLED_DrawPoint(x + a, y + b);

        OLED_DrawPoint(x + b, y + a);
        OLED_DrawPoint(x + b, y - a);
        OLED_DrawPoint(x - b, y - a);
        OLED_DrawPoint(x - b, y + a);

        a++;
        num = (a * a + b * b) - r*r;//计算画的点离圆心的距离
        if(num > 0)
        {
            b--;
            a--;
        }
    }
}

            for(n=0;n<8;n++)
