## STM32

1. map映射

   ```c
   /**
    * @Name    map
    * @brief   数据映射(缩放)
    * @param   - val: [输入]		数据采集值
    *          - I_Min: [输入]	数据最小值
    *          - I_Max: [输入]	数据最大值
    *          - O_Min: [输入]	需求最小值
    *          - O_Max: [输入]	需求最小值
    * @retval	需求处理值
    * @example map(128,0,4096,0.0,3.3) //0.10
    *			ADC采集引脚电压
    **/
   float map(float val, float I_Min, float I_Max, float O_Min, float O_Max)
   {
       return(((val-I_Min)*((O_Max-O_Min)/(I_Max-I_Min)))+O_Min);
   }
   ```
